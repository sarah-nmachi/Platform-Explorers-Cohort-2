# Creating agents with copilot chat

## Scenario:  
Your IT team wants a lightweight helper that Exchange administrators can talk to when they need a quick reminder about how to:
- Access the Exchange Admin Center
- Manage a user mailbox
- Manage distribution groups
- Check basic mail flow guidance
Instead of searching online, admins want a small Copilot agent that responds immediately with short explanations.

## Learning Objectives  
- Use Copilot Chat → New Agent to auto‑generate an agent
- Provide a natural‑language description to shape the agent
- Understand how the agent is created from a prompt

## Hands‑On Labs  
- Create the Agent Using Copilot Chat
- Review the Agent Auto‑Generated by Copilot
- Test using three Ultra‑Simple Topics
