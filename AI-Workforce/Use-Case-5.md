# Knowledge Source for Copilot agents

## Scenario:
The HR Team at Fabrikam wants an agent that answers basic HR questions—vacation policy, work hours, benefits, dress code, etc. All answers must come only from the company’s Employee Handbook, stored as a PDF/.docx in a SharePoint Online site.

## Learning Objectives:
- Create a simple agent using copilot studiop
- Set the knowledge source of the agent

## Hands-on Labs
- Create the agent in copilot studio
- Add the Knowledge Source (SharePoint File)
- Configure Response Behavior
- Test the Agent
